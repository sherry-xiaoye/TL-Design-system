# Spacing

## 当前状态

当前上传的 JSON 中未提供 Spacing Token。因此间距、组件高度、行高、圆角、阴影均为待定，不能伪造成 Token。

## 建议规则（非 Token）

以下为后台系统常见间距建议，仅作为设计确认前的占位：

| 场景 | 建议 | 状态 |
|---|---:|---|
| 小间距 | 4px / 8px | 待定 |
| 表单项间距 | 16px | 待定 |
| 筛选区与表格间距 | 16px | 待定 |
| 页面内边距 | 24px | 待定 |
| 区块间距 | 16px / 24px | 待定 |

## 禁止事项

- 禁止把上述建议值写入 `tokens.css` 作为已确认 Token。
- 禁止随意使用非系统间距。
- 禁止为了当前页面好看临时创建 spacing Token。
