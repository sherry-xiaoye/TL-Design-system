# Dashboard Page

Figma example:

- `分析页`
- `Charts` section

## Standard Structure

```text
Body
  Optional Filter-Bar
  Metric cards
  Chart grid
  Detail table or ranked list
```

## Chart Components

Observed chart patterns:

- `重要指标`
- `Area-Chart`
- `Line-Chart`
- `Bar-Chart`
- `Column-Chart`
- `Donut-Chart`
- `Tooltip`

## Typography

- Large metric: `Chart/大字号`
- Primary chart text: `Chart/主要文本`
- Section title: `Heading/页面内容的标题`
- Supporting text: `Body/次文本`

## Layout

- Use 1220px body content width.
- Card grid can use 3 columns with 20px gutters.
- Keep charts aligned to the same grid as tables and filter bars.

## Rules

- Dashboard charts should answer operational questions.
- Avoid decorative charts without labels, legends, or units.
- Use chart semantic tokens for series colors.
- Provide empty and loading states.
