## Usage Rules

- Use the Figma `Icon` names as the canonical source for icon naming.
- Prefer 16px icons for table actions, form controls, menu items, and toolbar utilities.
- Use 12px icons only in compact navigation affordances.
- Use 24px or larger only for empty states, result states, and dashboard feature visuals.
- Icon color should use semantic icon tokens: `Icon/Default`, `Icon/Secondary`, `Icon/Disabled`, `Icon/Brand`, `Icon/Success`, `Icon/Warning`, `Icon/Error`.
- Do not import unrelated icon packs unless a matching Figma icon is unavailable.
