## Layout Spacing Rules

- Form field vertical gap: use 20px or 24px depending on density.
- Inline control gap: use 8px or 12px.
- Toolbar button gap: use 8px.
- Card inner padding: use 16px.
- Page body padding: use 20px.
- Table cell horizontal padding: use 12px or 16px.
- Do not use arbitrary spacing that breaks the 4px rhythm.
