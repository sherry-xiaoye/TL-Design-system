# Grid Guidelines

## Desktop Grid

| Item | Value |
|---|---:|
| Reference viewport | 1440 x 900 |
| Header height | 56 |
| Sidebar width | 180 |
| Body width | 1260 |
| Body height | 844 |
| Body padding | 20 |
| Content width | 1220 |

## Form Grid

- Single centered form: 480px field width.
- Complex form: 360px columns.
- Label height: 18px.
- Control height: 36px.
- Form item vertical rhythm: 20px or 24px.

## Table Grid

- FilterBar width: 1220px.
- TableHeaderActions width: 1180px or 1220px depending container padding.
- Table body follows same left/right alignment as toolbar.

## Card Grid

- Three-card grid in a 1220px body can use around 393px cards with 20px gutters.
- Card padding: 16px.
- Card header height: 56px when using header slot.

## Rules

- Align filter, toolbar, table, pagination, and cards to the same content grid.
- Avoid arbitrary widths that make table columns and form labels drift.
- Use consistent gutters across sibling content blocks.
